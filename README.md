# 朝日朝食｜冰品網站

這是一個介紹天然水果製作的夏日冰品的靜態網站，包含各式冰品展示與購物待買清單功能，透過 GitHub Pages 部署與呈現。

🔗 [點我前往網站](https://shan11111111.github.io/web.github.io/home.htm)

---

## 📌 特色功能

- 🍧 多種天然水果冰品展示
- 🛒 待買清單功能（可新增商品並計算總金額）
- 📱 響應式網頁設計（適應不同裝置）
- 🎨 簡約清新的 UI 風格，強調圖片與商品陳列

---

## 🛠️ 使用技術

- HTML5
- CSS3
- JavaScript（原生 JavaScript 控制互動）
- GitHub Pages（作為部署平台）

---

## 🚀 快速開始

1. 前往 [網站連結](https://shan11111111.github.io/web.github.io/home.htm) 直接預覽使用。
2. 或是下載原始碼後使用本機瀏覽器開啟 `home.htm` 頁面即可。

```bash
git clone https://github.com/shan11111111/web.github.io.git
cd web.github.io
start home.htm  # Windows 使用者
# 或 open home.htm on macOS
```
